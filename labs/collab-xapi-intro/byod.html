This lab can be completed from any platform with any HTML5-compatible browser (except Internet Explorer if you're planning to connect to a cloud-registered device).

You'll also need local/VPN network connectivity to a Cisco Collaboration device, and appropriate 'admin' or 'integrator' role credentials.

Your PC must be equipped with both SSH and REST clients, as well as a Node.js runtime.

Recommended:

  • SSH: Mac/Linux: (built-in), Windows: Putty
  • REST API test client: cross-platform: Postman
  • Node.js: v8+ Node.js
  • cURL: Mac/Linux: (built-in), Windows: cURL
