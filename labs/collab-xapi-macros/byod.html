This lab can be completed from any platform with any HTML5-compatible browser (except Internet Explorer if you're planning to connect to a cloud-registered device).

You'll also need local/VPN network connectivity to a Cisco Collaboration device, and local device administration web UI 'admin' user credentials (on-premise) or admin user for Webex Control Hub (cloud-registered.)

Your PC must be equipped with a SSH and REST API test clients.

Recommended:

  • SSH: Mac/Linux: (built-in), Windows: Putty
  • REST API test client: cross-platform: Postman
